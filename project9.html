Extinct Animals Web Application

This team project was developed using Node.js and Express. We also used Bulma and CanvasJS for styling and interactive data visualization. The goal of this project was to create a web application that can retrieve data stored in a database. The topic we came up with was an extinct animals web application where users can view records on extinct animals and see data visualizations. The tables retrieved data directly from our database in MySQL.

The biggest takeaway for me is learning how to retrieve data on the front end from the back end by using routers and controllers.
